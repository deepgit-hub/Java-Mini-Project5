# Employee Salary and Deduction System

This is a Java console-based application that calculates employee salaries by applying deductions based on late days and leave days.

## Features
- Accepts multiple employees
- Takes basic salary, late days, and leave days as input
- Applies deductions for late arrivals and leaves
- Calculates final salary for each employee
- Generates a salary report including:
  - Total employees
  - Total salary paid
  - Highest salary
  - Lowest salary
  - Number of employees with fines

## Salary Rules
- Late day fine: ₹200 per day
- Leave day deduction: ₹500 per day
- Final Salary = Basic Salary − Total Deductions

## Concepts Used
- Java Scanner
- Arrays
- for loops
- Conditional statements
- Counters and accumulators

This project is built using only core Java fundamentals and focuses on improving logical thinking and real-world problem-solving without using methods, OOP concepts, or collections.
